# Sistema de Cadastro de Filmes em Java

Projeto simples desenvolvido em Java para gerenciamento de filmes através de um sistema de console.

## Funcionalidades

O sistema permite:

- Adicionar filmes
- Listar filmes cadastrados
- Pesquisar filmes por título
- Editar informações de filmes
- Excluir filmes

## Tecnologias utilizadas

- Java
- Programação Orientada a Objetos (POO)
- ArrayList
- Scanner para entrada de dados

## Estrutura do projeto

O sistema possui:

- Classe `Filme` → representa os dados do filme
- Classe `SistemaCadastroFilmes` → gerencia as operações
- Classe `ProgramaJava` → contém o menu principal
